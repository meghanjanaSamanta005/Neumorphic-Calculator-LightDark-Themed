## ✨ Features

* **Neumorphic Design:** Soft, organic UI using layered `box-shadow` for a realistic look.
* **Dark/Light Mode:** Includes a functional toggle switch for seamless theme switching.
* **Real-time Logic:** Fully functional arithmetic operations (Add, Subtract, Multiply, Divide).
* **Tactile Feedback:** Subtle "pressed" animations on button clicks for a physical feel.
* **Responsive:** Optimized for different screen sizes and devices.

## 🛠️ Technologies Used

* **HTML5:** Semantic structure for the calculator layout.
* **CSS3:** Custom properties (variables), Flexbox, Grid, and advanced shadow effects.
* **JavaScript:** DOM manipulation and core calculation logic.
